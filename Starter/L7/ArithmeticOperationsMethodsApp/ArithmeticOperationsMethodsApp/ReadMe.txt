+ Используя Visual Studio, создайте проект по шаблону ConsoleApplication.  
+ Создайте четыре метода для выполнения арифметических операций, с именами: 
      + Add – сложение, 
	  + Sub – вычитание,
	  + Mul – умножение, 
	  + Div – деление. 
+ Каждый метод должен принимать два целочисленных аргумента и выводить на экран 
результат выполнения арифметической операции соответствующей имени метода. 
- Метод деления Div, должен выполнять проверку попытки деления на ноль.  
+ Требуется предоставить пользователю возможность вводить с клавиатуры значения
операндов и знак арифметической операции, для выполнения вычислений.
