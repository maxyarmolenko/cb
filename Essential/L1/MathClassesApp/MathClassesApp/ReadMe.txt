+ Используя Visual Studio, создайте проект по шаблону Console Application.  
+ Создать 2 класса: Point и Figure.  
+ В классе Point создаем  два целочисленных поля (int) и одно строковое поле (string) c модификатором доступа private. 
+ В классе Point создаем три свойства c модификатором доступа public с одним методом доступа get для 2 целочисленных полей (int) 
и одного строкового поля (string).  
+ В классе Point создаем пользовательский конструктор (который принимает аргументы), в теле которого 
инициализируем поля значениями аргументов.
+ В классе Figure создаем конструкторы, которые принимают от 3-х до 5-ти аргументов типа Point. 
+ В классе Figure создаем два метода: 
         + double LengthSide(Point A, Point B) (функция с возращающим значением типа double),
		   который рассчитывает длину стороны многоугольника;                                 LengthSide = √(xb - xa)2 + (yb - ya)2
          + void PerimeterCalculator(), который рассчитывает периметр многоугольника по формуле, P = (a + b) * 2 ;
+ Написать программу, которая выводит на экран название и периметр многоугольника. 
Название многоугольника выводится в классе Figure помощи поля type    в зависимости от колличества элементов массива, если 3 - трехугольник, 4 - квадрат, ромб или прямоугольник
